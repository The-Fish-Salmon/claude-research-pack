# 00_Inbox

The **drop zone** for unsorted notes. Anything you (or a skill) writes here is implicitly "needs review before it counts as part of the vault".

## What lives here

- Drafts written by the `deep-research` skill at the end of a `lit-review` or `systematic-review` run (file pattern: `research-<slug>-<date>.md`).
- Quick captures from the user (idea fragments, paper mentions to triage later).
- Anything you're not sure where to put yet.

## Promotion

Don't read from `00_Inbox/` as if it were canonical. Periodically curate:

- Promote vetted research drafts → move to `10_Projects/<project>/` or `30_Literature/`.
- Promote ideas → flesh out and move to `20_Areas/`.
- Drop dead ends.

## Conventions

- The `paper-capture` skill does **not** write here — it goes straight to `30_Literature/` because metadata is mechanically resolved and de-duped.
- The `deep-research` skill **always** writes its draft here, never directly to `30_Literature/` or project folders. Curation is a human step.
